# Sales Forecasting using Superstore Dataset

## Project Overview
This project analyzes Superstore sales data to identify sales trends, top-performing states, category-wise sales distribution, and the relationship between sales, profit, quantity, and discount using Python.

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Features
- Data Cleaning and Preprocessing
- Monthly Sales Trend Analysis
- Top 10 States by Sales Analysis
- Category Wise Sales Distribution
- Correlation Analysis
- Data Visualization

## Dataset
Superstore Dataset containing information about orders, customers, products, sales, profit, and discounts.

## Key Skills
- Data Analysis
- Data Visualization
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Python Programming

## Conclusion
This project helped in understanding sales patterns, customer trends, and factors affecting profit and sales performance.
